Credits
       Hey there! My name's Rishi. I'm the creator of this game, and it was originally made for an Engineering school project. I started making this game my freshman year, and the original version was around ~650-700 lines of code in Python. However, I decided to remake the game in JavaScript, and add a nice user interface in order to make it more appealing.

       Honestly, I made this game in two days, but I spent a lot of time thinking about how to make the game more unique; I decided to add achievements, levels, and more customization. The original game was a bit boring; sure, you could beat four various AI's, but you couldn't really do much else. There was a lack of replayability, and I wanted to fix that.

       I hope that you enjoyed the game as much as I enjoyed making it. I plan on updating the game every so often. And plus, I'll probably get wayyy better at coding in the next couple of years (hopefully). Maybe I'll even make a sequel to this game, who knows? For now, fellow gamer, I bid you farewell.

- R. Santhanam
